# Editing this page

Everything lives in one file: `index.html`. You can edit it directly on GitHub (click the pencil icon on the file) — no coding needed.

## 1. Photos
Already set up — `dan.png` (profile) and `background_1.png` (background) are in this folder and referenced in `index.html`. To swap them later, replace the files (keep the same names) or update the `src="..."` in `index.html`.

## 2. Add your links
Open `index.html`, find each placeholder below, and replace it with your real link (keep the quotes):

| Placeholder | Replace with |
|---|---|
| `PASTE_SPOTIFY_LEAD_SINGLE_LINK_HERE` | Spotify link — lead single |
| `PASTE_YOUTUBE_LEAD_SINGLE_LINK_HERE` | YouTube link — lead single |
| `PASTE_SPOTIFY_SECOND_SINGLE_LINK_HERE` | Spotify link — second single |
| `PASTE_YOUTUBE_SECOND_SINGLE_LINK_HERE` | YouTube link — second single |
| `PASTE_SPOTIFY_EP_LINK_HERE` | Spotify link — full EP |
| `PASTE_YOUTUBE_EP_LINK_HERE` | YouTube link — full EP |
| `PASTE_INSTAGRAM_LINK_HERE` | Your Instagram profile |
| `PASTE_FACEBOOK_LINK_HERE` | Your Facebook page |
| `PASTE_YOUTUBE_CHANNEL_LINK_HERE` | Your YouTube channel |
| `PASTE_SPOTIFY_ARTIST_LINK_HERE` | Your Spotify artist profile |

Also swap the placeholder text `Track Name` / `EP Name` in the two `<span class="link-label">` lines for the real titles.

## 3. Publish changes
Any edit you make on github.com and commit will go live automatically within a minute or two (GitHub Pages redeploys on every push).
